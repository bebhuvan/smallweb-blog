# Indie Blog Aggregator - Implementation Spec

## Overview
A curated, beautiful RSS aggregator for indie blogs - hand-picked and categorized.
An "indie digital garden" celebrating independent voices on the web.

---

## Tech Stack

| Component | Technology | Purpose |
|-----------|------------|---------|
| Framework | Astro | Static site generation, content-focused |
| Styling | Tailwind CSS | Typography-first, responsive design |
| Deployment | Cloudflare Pages | Free tier, global CDN |
| Feed Refresh | GitHub Actions | Scheduled twice daily |
| Data Storage | JSON files | No database needed for V1 |

---

## Project Structure

```
/
├── src/
│   ├── pages/
│   │   ├── index.astro          → Main feed (all posts)
│   │   ├── category/[slug].astro → Category view
│   │   └── blog/[slug].astro    → Single blog's posts
│   ├── components/
│   │   ├── PostCard.astro       → Individual post display
│   │   ├── Header.astro         → Navigation + categories
│   │   ├── Footer.astro
│   │   └── ThemeToggle.astro    → Light/dark mode switch
│   ├── layouts/
│   │   └── Base.astro           → HTML shell, fonts, meta
│   └── styles/
│       └── global.css           → Base typography, Tailwind
├── data/
│   ├── blogs.json               → Curated blog list (you edit this)
│   └── cache/
│       └── posts.json           → Generated by feed fetcher
├── scripts/
│   └── fetch-feeds.js           → RSS fetcher script
├── .github/
│   └── workflows/
│       └── refresh.yml          → Scheduled feed refresh
└── astro.config.mjs
```

## Data Schemas

### blogs.json
```json
{
  "blogs": [
    {
      "id": "paul-graham",
      "name": "Paul Graham",
      "url": "https://paulgraham.com",
      "feed": "https://paulgraham.com/rss.html",
      "categories": ["tech", "philosophy"],
      "description": "Essays on startups and life"
    }
  ]
}
```

### posts.json (generated)
```json
{
  "lastUpdated": "2024-01-15T12:00:00Z",
  "posts": [
    {
      "id": "unique-post-id",
      "blogId": "paul-graham",
      "title": "Post Title",
      "link": "https://paulgraham.com/post.html",
      "date": "2024-01-15T00:00:00Z",
      "excerpt": "First 200 characters of content..."
    }
  ]
}
```

## Design System

### Color Palette

**Light Mode:**
```css
--text:       #1a1a1a
--text-muted: #666666
--bg:         #fafafa
--card-bg:    #ffffff
--accent:     #0066cc
--border:     #e5e5e5
```

**Dark Mode:**
```css
--text:       #e5e5e5
--text-muted: #999999
--bg:         #0f0f0f
--card-bg:    #1a1a1a
--accent:     #66b3ff
```

### Typography
- Body: Inter / System fonts
- Headings: Clean sans-serif
- Generous whitespace, typography-focused

### Design Principles
- Typography-first
- Generous whitespace
- Muted palette
- No visual noise
- Inspired by: Swiss design, Hacker News, print aesthetics

## Core Features (V1)

1. **Curated Blog Registry** - Manual `blogs.json` editing
2. **Automated Feed Aggregation** - GitHub Actions twice daily
3. **Category Navigation** - Clean filter UI, category pages
4. **Beautiful Feed UI** - Typography-focused cards
5. **Individual Blog Profiles** - `/blog/[slug]` pages
6. **Dark/Light Mode** - Toggle with localStorage persistence

---

## GitHub Action Workflow

Runs twice daily:
1. Checkout repo
2. Run `fetch-feeds.js`
3. Commit updated `posts.json`
4. Trigger Cloudflare Pages rebuild
